Bienvenidos a mi proyecto

Start

Mi nombre es Franco Osores, soy un desarrollador Frontend, de Tucumán Argentina. Este proyecto fue creado en 48Hrs para la empresa DEVSU como parte de un challenge técnico. Antes de comenzar me gustaría hacer algunas aclaraciones:

  • El proyecto no es responsive ya que no era solicitado en las especificaciones
  • No se utilizó ningún framework para los estilos
  • Por lo anterior cada componente fue creado a medida para esta app, desde los Buttons hasta los Formfields e Inputs
  • El diseño se basó en un png brindado para el challenge.
  • Por cuestiones de tiempo en el customizado de los componentes no se pudo realizar testing unitario
  • Algunas de las dificultades que encontré al abordar el proyecto:
    • Copiar los estilos sin las especificaciones fue dificil, sin embargo en cuanto a estilos quedé bastante conforme
    • Al ser componentes 100% custom se dificultó la comunicación entre los mismos, pero se consiguió que la app funcionará como se esperaba

Muchas gracias por la oportunidad

Me divertí mucho trabajando en el proyecto
